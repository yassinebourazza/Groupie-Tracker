Groupie Tracker
is a web application written in Go that fetches and displays data from an external API about music artists. The website presents detailed information for each artist, including images, members, concert locations, and album history.

This project interacts with a provided API that contains structured information about music artists. The goal is to create a complete and user-friendly website that shows: - Artist name and picture - Year the artist started - Date of their first album - Band members - Locations of past and future concerts The app parses and organizes this data, then dynamically displays it on a styled web page using HTML and CSS.
